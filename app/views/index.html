Application Info

This App is built on Node.JS and it uses Angular, MongoDB, Express, Facebook Graph API, XMPP, Cron. If you are interested, here's the code on GitHub. (github image) What this app is great for: If you have a boring day in front of you, why not take a few moments to write thoughtful messages to your friends for their birthdays and schedule to send it to them on their birthday. This app will send a personal message using Facebook's chat functionality to any of your friends. It's great for never forgetting to remember another person's birthday. (chat)

Future Functionality

1. You can manually add a person's birthday on here if they don't already have it. (database insertion).
2.Subscription to e-mail functionality reminding you everyday whose birthday it is. (e-mail)
3. Ability to cancel future jobs. (cancel logo)
4. Ability to edit future messages.
